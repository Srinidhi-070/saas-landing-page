## 🧪 **Getting Started (Dev Mode)**

```bash
# 1. Clone this repo
git clone https://github.com/yourusername/saas-landing-page.git

# 2. Navigate into the folder
cd saas-landing-page

# 3. Install dependencies
npm install

# 4. Run the dev server
npm run dev
```
